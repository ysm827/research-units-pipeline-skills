# Source Tutorial Spec Excerpt

## Audience

- Early-stage robotics and machine learning readers who want a practical
  introduction to robot learning pipelines.

## Prerequisites

- Basic Python
- Introductory linear algebra
- Very light familiarity with supervised learning

## Learning Objectives

- Explain the interaction loop behind reinforcement learning.
- Distinguish general reinforcement-learning concepts from robot-learning
  implementation stacks.
- Identify why modern robot-learning tutorials combine concepts, data
  pipelines, and tooling.
- Describe one advanced representation idea, using SE(3)-equivariance as an
  example of inductive bias.

## Delivery

- Reader-first tutorial article
- Article PDF
- Beamer slides for presentation and lightweight self-study
